# Key Features & Implemented Technologies

  * **Network Optimization:** VLAN, OSPF, and Static Routing
  * **Secure Connectivity:** IPSec VPN and NAT
  * **Security & Access Control:** ACLs and Network Security
  * **Modern Integration:** The Impact of IoT Devices on Corporate Networks
  * **Advanced Architecture:** DMZ, Segmentation, and Scalability


# Technical Requirements & Compatibility

    [!IMPORTANT]
    Recommended Version: Cisco Packet Tracer 8.2.2.
    Compatibility Note: This topology was developed and tested in version 8.2.2 to ensure full compatibility with all used IoT devices and configurations. Opening the file in newer versions (e.g., 9.0.0) may cause 
    unexpected behavior or loss of some IoT functionalities.

    Performance: Due to the high complexity of the network, the file may take 1-2 minutes to load and converge. Please wait until all link lights turn green (OSPF/STP convergence) before testing connectivity.
