# Professional-Corporate-Network-Simulation
Designing a complex corporate network topology using Cisco Packet Tracer simulation tool.
The network follows the basic principles and several types of configurations that I have implemented. The network is divided into
a central part and a branch office, which are located far from each other, but are fully interconnected via an encrypted
IPSec VPN tunnel through two external ISPs. It also includes a DMZ server area and a
4G mobile network. This project demonstrates that even in a simulator like Packet Tracer, it is possible to create a fully
functional, secure, and professionally designed network topology that connects the central and
remote locations of a company.
